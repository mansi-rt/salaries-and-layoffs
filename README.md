# salaries-and-layoffs
A project to visualize median salaries and explore their data along with studying the layoff patterns in 2020-2025 for big tech and major companies.

Data Source Layoffs:
https://www.kaggle.com/datasets/swaptr/layoffs-2022

Data Source Salaries:
https://www.kaggle.com/datasets/hummaamqaasim/jobs-in-data?select=jobs_in_data.csv
